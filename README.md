## How to run:
Clone repo:
```
$ git clone git@github.com:mfmiller7/beer.git
$ cd beer
```
Start the server:
```
$ cd backend
$ node server.js
```
Run the frontend:
```
$ cd backend
$ node server.js
$ cd frontend
$ rm -r node-modules
$ npm install
$ npm run dev
```
