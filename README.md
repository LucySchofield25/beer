## How to run:
In your terminal, clone repo and start the server
```
git clone git@github.com:mfmiller7/beer.git
cd beer/backend
node server.js
```
In a new tab, run the frontend
```
cd beer/frontend
npm i @react-oauth/google
npm run dev
```
